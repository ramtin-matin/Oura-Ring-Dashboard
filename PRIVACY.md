# Privacy Policy

## Oura Ring Dashboard

Oura Ring Dashboard is a personal, open-source software project created for educational and personal use.

The application is intended to access and analyze only the developer's own Oura Ring data. It is not currently offered as a public service for other users.

Oura data may be stored locally in a private PostgreSQL database running on the developer's computer. The application does not sell, rent, share, or distribute Oura data to third parties.

Authentication credentials, access tokens, refresh tokens, and environment variables are not intentionally committed to the public GitHub repository.

This application is not affiliated with, endorsed by, or operated by Oura Health Oy.
